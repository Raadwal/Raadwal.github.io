Bubble Sort

Basics Information

Brief Description

Bubble sort (sometimes referred to as sinking sort) is one of the simplest sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order. The pass thorugh the list is repeated until the lsit is sorted.

How the code looks like?

Bubble Sort implementation in python:
def bubbleSort(array):
n = len(array)
for i in range(n - 1):
for j in range(0, n - i - 1):
if array[j] > array[j + 1]:
swap(array[j], array[j + 1])

Is it efficient?

Time Complexity Memory Complexity
Best O(n) O(1)
Avarage O(n2) O(1)
Worst O(n2) O(1)
Selection Sort

Basics Information

Brief Description

Selection sort is an in-place comparison sorting algorithm. Selection sort is noted for its simplicity and has performance advantages over more complicated algorithms in certain situations, particularly where auxiliary memory is limited.

How the code looks like?

Selection Sort implementation in python:

Insertion Sort

Basics Information

Brief Description

Insertion sort is a simple sorting algorithm that builds the final sorted array (or list) one item at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

How the code looks like?

Insertion Sort implementation in python:
def insertionSort(array):
for step in range(1, len(array)):
key = array[step]
j = step - 1
while(j >= 0 and key &; array[j]):
array[j + 1] = array[j]
j = j - 1
array[j + 1] = key

Is it efficient?

Time Complexity Memory Complexity
Best O(n) O(1)
Avarage O(n2) O(1)
Worst O(n2) O(1)
Shell Sort

Basics Information

Brief Description

Shell sort is an in-place comparison sort. It can be seen as either a generalization of sorting by exchange (bubble sort) or sorting by insertion (insertion sort). The method starts by sorting pairs of elements far apart from each other, then progressively reducing the gap between elements to be compared. By starting with far apart elements, it can move some out-of-place elements into position faster than a simple nearest neighbor exchange. The running time of Shellsort is heavily dependent on the gap sequence it uses. For many practical variants, determining their time complexity remains an open problem.

Quick Sort

Basics Information

Brief Description

Quick sort Quicksort is a divide-and-conquer algorithm. It works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays, according to whether they are less than or greater than the pivot. For this reason, it is sometimes called partition-exchange sort. The sub-arrays are then sorted recursively. This can be done in-place, requiring small additional amounts of memory to perform the sorting.

How the code looks like?

Quick Sort implementation in python:
def partition(arrray, low, high):
pivot = array[high]
i = low - 1
for j in range(low, high):
if array[j] <= pivot:
i = i + 1
swap(array[i], array[j])
swap(array[i + 1], array[high])
return i + 1

def quickSort(array, low, high):
if low < high:
pi = partition(array, low, high)
quickSort(array, low, pi - 1)
quickSort(array, pi + 1, high)

Is it efficient?

Time Complexity Memory Complexity
Best O(n log n) O(log n)
Avarage O(n log n) O(log n)
Worst O(n2) O(log n)

Bogo Sort

Basics Information

Brief Description

Bogo sort (also known as permutation sort, stupid sort, or slow sort) is a highly inefficient sorting algorithm based on the generate and test paradigm. The function successively generates permutations of its input until it finds one that is sorted. It is not useful for sorting, but may be used for educational purposes, to contrast it with more efficient algorithms.
